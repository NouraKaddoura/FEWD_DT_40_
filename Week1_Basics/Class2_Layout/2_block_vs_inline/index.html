Types of CSS Boxes: Block, Inline, Inline-block:
  • Block: a block box is a box that's stacked upon other boxes - It has a break between ther content. Starts on it's own line
  • Inline: an inline box is the opposite, it flows with the document's text. It will appear on the same line as other elements.
  • Inline-block: in between the first two. It can be sized using width and height.
